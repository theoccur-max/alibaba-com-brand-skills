1. folder or item link

4. root Buyer library link
5. file or file-share link only when the user explicitly asks for a direct file link or when no browseable folder/category link is available

- the exact known asset title
- the category path
- the preferred browseable Done link from this file
- a short note about whether the link opens a folder, category, or file
